### Directrices de Ejecución

1. **Prerequisitos:**
   - Node.js 18+ + pnpm 9.x: ✅ instalados
   - PostgreSQL local o Neon branch: ✅
   - Vitest + Playwright paquetes: ✅

2. **Ejecución (local development):**
   ```bash
   pnpm test:unit          # Run all unit tests
   pnpm test:integration   # Run API integration (needs server running)
   pnpm test:e2e           # Run E2E Playwright (needs server at :3000)
   pnpm test:coverage      # Generate coverage report (80% min check)
   ```

3. **CI/CD (automatic on push/PR):**
   - GitHub Actions ejecutará .github/workflows/ci.yml
   - Jobs: Lint → Unit → Integration → E2E → Build → Summary
   - Artifacts: playwright-report, coverage/** uploaded to Codecov
